## 🚀 Getting Started

### 1️⃣ Clone the Repository
```sh
git clone https://github.com/YOUR_USERNAME/portfolio-website.git
cd portfolio-website
```

### 2️⃣ Run Locally
Simply open `index.html` in your browser.

### 3️⃣ Deployment
Deployed using Firebase Hosting:
```sh
firebase init
firebase deploy
```
